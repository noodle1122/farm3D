# 天台农场 - 3D高斯泼溅模型展示

一个基于3D Gaussian Splatting技术的天台农场展示网站，让您在浏览器中沉浸式体验城市农业的魅力。

## 🌱 项目简介

天台农场是一个结合现代Web技术和3D可视化技术的创新项目。通过3D Gaussian Splatting技术，我们将真实的天台农场以高精度3D模型的形式呈现，用户可以在浏览器中进行360度自由视角探索。

### ✨ 主要功能

- **3D模型交互**：支持鼠标拖拽旋转、平移和缩放操作
- **响应式设计**：适配桌面和移动设备
- **流畅动画**：加载动画、悬停效果和页面过渡
- **信息展示**：农场照片画廊和详细介绍
- **现代UI**：使用Tailwind CSS构建的美观界面

## 🚀 技术栈

- **3D渲染**：[Gaussian Splats 3D Viewer](https://github.com/mkkellogg/GaussianSplats3D)
- **前端框架**：原生JavaScript + HTML5 + CSS3
- **样式框架**：Tailwind CSS
- **图标库**：Font Awesome
- **字体**：Noto Sans SC / Noto Serif SC

## 🛠️ 配置说明

### 模型配置
在 `index.html` 中的 `modelConfig` 对象可以调整以下参数：

```javascript
const modelConfig = {
  modelName: 'demo.splat',        // 模型文件名
  cameraPosition: [0, 2, 5],      // 初始相机位置
  cameraLookAt: [0, 0, 0],        // 相机看向的目标点
  cameraUp: [0, -1, 0],          // 相机向上方向
  modelRotation: [1, 0, 0, 0],    // 模型旋转（四元数）
  modelScale: [1, 1, 1]          // 模型缩放
};
```

### 相机设置说明
- 如果模型显示颠倒，尝试修改 `cameraUp` 为 `[0, 1, 0]`
- 使用 `modelRotation` 调整模型方向：
  - 绕X轴旋转180度：`[1, 0, 0, 0]`
  - 绕Y轴旋转180度：`[0, 1, 0, 0]`
  - 绕Z轴旋转180度：`[0, 0, 1, 0]`
  - 无旋转：`[0, 0, 0, 1]`

## 🚀 部署说明

1. 确保所有文件在同一目录下
2. 需要一个支持CORS的Web服务器
3. 模型文件 `demo.splat` 必须存在且可访问
4. 建议使用现代浏览器（Chrome、Firefox、Safari、Edge）

## 📝 注意事项

- 3D模型文件较大，首次加载可能需要一些时间
- 确保服务器配置正确的MIME类型
- 移动设备需要支持WebGL
- 如果遇到跨域问题，请配置服务器CORS策略
